== Description ==  
LearnPress Stripe is an extension plugin to provide feature to use Stripe for selling courses in LearnPress

== Installation ==

**From your WordPress dashboard**  
1. Visit 'Plugin > Add new'.  
2. Search for 'LearnPress Stripe'.  
3. Activate LearnPress from your Plugins page.  

**From WordPress.org**  
1. Search, select and download LearnPress Stripe.  
2. Activate the plugin through the 'Plugins' menu in WordPress Dashboard.
